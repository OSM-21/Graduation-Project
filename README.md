# Jazan Smart Chatbot (Local, Arabic-First)

بوت عربي محلي للمستجدين — فيه:
- قاعدة معرفة (JSON) + تدريب TF‑IDF (مع خيار تعلّم تمثيلات جمل Sentence Transformers).
- واجهة محادثة عربية مع كتابة متدرّجة، رموز تعبيرية، صوت (TTS) واستقبال صوت (STT — متصفح).
- تسجيل مستخدمين، أسئلة أمان، أدوار (Admin / Editor / User).
- لوحة أدمن لإدارة الأسئلة، الأسئلة غير المعروفة، والإحصائيات.
- نظام تقييم 👍 👎، ونسخ احتياطي تلقائي لقاعدة البيانات.

## التشغيل السريع
```bash
# 1) داخل المجلد
pip install -r requirements.txt

# 2) إعداد موارد NLTK (مرة واحدة)
python -c "import nltk; nltk.download('punkt'); nltk.download('punkt_tab')"

# 3) تشغيل
python run.py
```
يفتح على: http://127.0.0.1:5000

> **ملاحظة:** إذا رغبت بتفعيل الفهم العميق للجمل، أبقِ `sentence-transformers` في المتطلبات كما هو.
سيتم استخدامه تلقائيًا إن توفر.

## هيكلة المشروع
- `app/` الكود (Flask + المنطق).
- `data/` قاعدة المعرفة، النماذج، والأسئلة غير المعروفة.
- `backups/` النسخ الاحتياطي.
- `run.py` نقطة التشغيل.

## مميزات رئيسية
- Self-Learning: أي سؤال غير مفهوم يُسجل في `data/unknown_questions.txt` وليستعرضه الأدمن ويضيف جوابه.
- Analytics: إحصائيات الاستخدام والتقييمات.
- TTS/STT: توليد صوتي للخروج، وإدخال صوتي عبر Web Speech API.
- Arabic NLP: إزالة كلمات شائعة + تهيئة نصوص.

## نسخ احتياطي
يُنشأ ملف نسخ احتياطي من قاعدة البيانات إلى `backups/backup_YYYYMMDD_HHMMSS.db` عند كل تشغيل.
